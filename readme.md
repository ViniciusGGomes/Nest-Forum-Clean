Este projeto é uma API de fórum desenvolvida com foco em **NestJS**, servindo como base para aplicar posteriormente conceitos como DDD, Clean Architecture e boas práticas de backend.

Este repositório representa a fase de fundação do projeto, onde o foco está na configuração correta do ambiente, entendimento do framework e preparação para evoluir a aplicação com uma arquitetura mais robusta.

## 🚧 Etapa atual

Nesta fase estou estruturando e configurando o ambiente da aplicação, entendendo profundamente o funcionamento do NestJS antes de avançar para regras de negócio mais complexas.

### ✅ O que já foi feito

- Setup inicial do **NestJS**
- Estudo e uso de **decorators** (controllers, modules, providers)
- Configuração do ambiente com **Docker Compose**
- Integração com **Prisma ORM**
- Configuração de variáveis de ambiente com `ConfigModule`
- Criação de controller para **criação de conta**
- Hash de senha antes de persistir no banco
- Validação de dados com **Zod + Pipe customizado**
- Testes de rota utilizando **REST Client (VSCode)**
